# Portfolio Website

> A modern, responsive personal portfolio website showcasing my skills, projects, experience, and journey as a Software Developer.

🌐 **Live Demo:** https://dev-sohail.github.io/Portfolio-Website/

---

## 📖 About

This portfolio website serves as my personal space on the web where I showcase my projects, technical skills, experience, and contact information.

The goal of this project is to provide recruiters, clients, and fellow developers with a quick overview of my work while demonstrating my frontend development skills through a clean and responsive design. :contentReference[oaicite:0]{index=0}

## ✨ Features

- 👋 Personal introduction
- 💼 Project showcase
- 👨‍💻 About Me
- 📞 Contact information

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/dev-sohail/Portfolio-Website.git
```

Navigate to the project

```bash
cd Portfolio-Website
```

Open

```text
index.html
```

or launch it using **VS Code Live Server**.
